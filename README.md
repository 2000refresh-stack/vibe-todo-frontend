# 📝 Todo React App

Vite + React + TypeScript로 구축된 모던한 할일 관리 애플리케이션입니다.

## ✨ 주요 기능

- ✅ 할일 추가 (작성자 이름 선택 가능)
- ✏️ 할일 수정
- 🗑️ 할일 삭제
- 📋 할일 목록 조회
- 🎨 모던하고 반응형 UI
- ⚡ 빠른 개발 환경 (Vite)
- 🔒 에러 처리 및 유효성 검증

## 🚀 시작하기

### 사전 요구사항

- Node.js 16.0 이상
- npm 또는 yarn
- **백엔드 서버가 `localhost:5000`에서 실행 중이어야 합니다**

### 설치 및 실행

```bash
# 의존성 설치 (이미 완료됨)
npm install

# 개발 서버 실행
npm run dev
```

개발 서버가 시작되면 브라우저에서 `http://localhost:5173`으로 접속하세요.

### 빌드

```bash
# 프로덕션 빌드
npm run build

# 빌드 결과 미리보기
npm run preview
```

## 🔌 백엔드 API 연동

이 앱은 다음 API 엔드포인트를 사용합니다:

- `GET /todos` - 모든 할일 조회
- `POST /todos` - 새 할일 생성
- `PATCH /todos/:id` - 할일 수정
- `DELETE /todos/:id` - 할일 삭제

### 환경변수 설정

백엔드 서버 주소는 환경변수로 관리됩니다. 프로젝트 루트에 `.env` 파일을 생성하고 다음과 같이 설정하세요:

```env
# 프로덕션 서버 (Heroku)
VITE_API_BASE_URL=https://vibe-todo-d37d1df139db.herokuapp.com

# 로컬 개발 서버를 사용하려면 아래 주소로 변경하세요
# VITE_API_BASE_URL=http://localhost:5000
```

**중요**: 
- Vite에서는 환경변수 이름이 `VITE_`로 시작해야 합니다
- `.env` 파일을 수정한 후에는 개발 서버를 재시작해야 합니다
- `.env` 파일은 Git에 커밋되지 않습니다 (보안상의 이유)

### 백엔드 서버 실행 방법 (로컬 개발 시)

로컬 백엔드를 사용하려면 `.env` 파일에서 주소를 변경하세요:

**터미널 1 (백엔드):**
```bash
cd C:\vibe-coding\todo-backend
node index.js
```

**터미널 2 (프론트엔드):**
```bash
cd C:\vibe-coding\todo-react
npm run dev
```

## 🛠️ 기술 스택

- **React 19** - UI 라이브러리
- **TypeScript** - 타입 안전성

## 📝 사용 방법

1. **할일 추가**: 상단 입력창에 할일 내용을 입력하고 작성자 이름(선택사항)을 입력한 후 "추가하기" 버튼을 클릭합니다.

2. **할일 수정**: 각 할일 항목의 "수정" 버튼을 클릭하면 수정 모드로 전환됩니다. 내용을 수정한 후 "저장" 버튼을 클릭합니다.

3. **할일 삭제**: 각 할일 항목의 "삭제" 버튼을 클릭하면 확인 메시지가 나타나고, 확인하면 삭제됩니다.

## ⚠️ 주의사항

- 백엔드 서버가 실행 중이지 않으면 에러 메시지가 표시됩니다.
- CORS 설정이 필요할 수 있습니다 (백엔드에서 설정).
- 할일 내용이 비어있으면 추가/수정할 수 없습니다.
- MongoDB가 실행 중이어야 백엔드 서버가 정상 작동합니다.
